# Social Robotics Group 4
This is a python script that runs an Alpha Mini robot. 
Using LLMs to communicate and play the *With Other Words* guessing game.

## Installation
To initallize the environment follow these steps: 
1. uv sync 
2. source .venv/bin/activate 

## API key
Create a .env file to hold your api key. 
The api key should be structured as in example.env. 

## Running the correct conditions
To run the code for Condition A, use:
'''
git switch condition_A
'''

To run the code for Condition B, use:
'''
git switch main
'''

## Running file
To run the file you can use the play button in the top right hand side of the screen, this will start the code.

Alternatively, run the following command in the terminal:

'''
python src/main.py
'''
